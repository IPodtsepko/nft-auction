# NFT аукцион

Проект подразумевает реализацию аукциона на покупку NFT. Для NFT предполагается использовать шаблон ERC721. Вопрос выбора типа аукциона пока открыт:

- Английский аукцион
- Скандинавский аукцион
- Голандский аукцион
- Аукцион первой цены
- Пенни аукцион
- Аукцион Викри
- Аукцион второй цены
